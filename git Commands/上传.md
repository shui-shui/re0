

# 操作



登录 GitHub



点击右上角 "+" → "New repository"



填写仓库名（比如 re0）



不要勾选 "Add a README file"



点击 Create repository



# 命令



git init

git config user.name "shui-shui"

git config user.email "3236837006@qq.com"

\# 4. 添加所有文件到暂存区

git add .

\# 5. 提交文件

git commit -m "首次提交"

\# 6. 关联远程仓库（替换成你的仓库地址）

git remote add origin https://github.com/你的用户名/re0.git

\# 7. 推送代码到 GitHub

git branch -M main

git push -u origin main
